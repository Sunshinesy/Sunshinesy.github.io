一门新的编程语言学习，大多以"hello world"开始，工作之余总结下PHP、JS、Python的"hello world"实现方式，总结与回顾下三种语言的基础知识。

一、PHP Hello World

1、最简单的方式直接在html文档中加入以下代码，即可输出"Hello World"

<?php echo "Hello World"; ?>

以上结果输出：

Hello World

2、第二种方式，若有完整的服务器框架，例如采用MVC框架模型输出"Hello World"，简单一点，采用VC模型即可，建立类与视图函数，再简单一点，在视图文件或者网页中，直接定义类与函数，然后通过类创建对象，调用函数，输出“Hello World"。以下上代码：

<?php 
class Hello{                                 //建立Hello 类
  public function World(){         //建立World() 函数
    return "Hello World";            //返回值"Hello World"
  }
}
$HW=new Hello;                           //通过Hello类创建一个对象$HW
$Hello=$HW->World();                //通过对象$HW调用类中的World() 函数，返回值赋给$Hello
echo $Hello;                                  //输出$Hello
?>

以上结果输出：

Hello World

二、JS Hello World

1、JS常用在网页交互上，因此使用alert()函数来生成JS网页弹窗是不错的选择：

<script>alert("Hello World")</script>

以上结果输出弹窗：

2、js函数编写完成后是可以直接使用的，因此可以直接写一个弹出"Hello World"的函数HW()，然后调用这个函数即可，下面上代码：

<script>
function HW(){
     alert("Hello World")
     }
HW();
</script>

以上结果输出：

三、Python Hello World

1、Python 可以简单的通过print()函数来输出"Hello World"：

# -*- coding:utf-8 -*-
print('Hello World')

运行Python代码，在Console中可看到输出结果：

Hello World

2、当然，Python的函数也是可以直接使用的，因此可以直接写一个函数HW()打印"Hello World"，然后调用这个函数即可，下面上代码：

# -*- coding:utf-8 -*-
def HW():
    print('Hello World')
if __name__ == '__main__':
    HW()

运行Python代码，在Console中可看到输出结果：

Hello World
